# Support

## Community support

- Open a GitHub issue in this repository with:
  - what you’re trying to do,
  - a minimal reproduction (requests + responses),
  - and (if relevant) your conformance report (`npm run conformance -- --out ...`).

## Commercial support

For hosted deployments, enterprise support, and compliance requests:
- `support@lam-protocol.com`

## Security

If you believe you’ve found a security vulnerability, please **do not** open a public issue.

Email:
- `support@lam-protocol.com` (subject: “Security: <short description>”)
